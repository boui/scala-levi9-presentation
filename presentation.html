I. LANGUAGE:FOR

If class has map and flatmap then we can use for comprehension to work with it

                   
                       for(i<- 1 to 20) print(i)
                       // (1 to 20) : Range.Inclusive
                   
                
                   
                       for(value<- valueOption)
                        print("will be printed out if i was set"+i)
                       // valueOption: Option[Int]
                   
                
                   
                       val lolypop =
                        for (f1<- future1, f2<- future2) yield f1+f2
                       //future1 and future2 :Future[Int]

I. LANGUAGE:FOR

Is more complex than you think

                   
                   for {
                       bound <- list
                       out <- f(bound)
                   } yield out
                   //is the same as
                   list flatMap f
                   for (bound <- list) yield f(bound)
                   //is the same as
                   list map f
